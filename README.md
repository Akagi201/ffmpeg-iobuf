ffmpeg-iobuf
============

Read and write from memory with FFmpeg.

## Features
* Read and write from memory with FFmpeg.
* Use cmake for cross-platform build.

## Dependencies
* libavformat
* libavcodec
* libavutil

## Build
* `./start_build.sh`
* build target is located in build/ffmpeg_iobuf

## Usage
* `ffmpeg_iobuf`
